How to use this theme

Used for the filename and the commands below: custom.json

  1. 1 Pick a preset above, then tweak any token until the live preview looks right.
  2. 2 Name your theme.
  3. 3 Click Copy for agent and paste it to your AI agent (with the cookiebite skill) — it applies the theme to that report. To make it your default for every report, use Set as my default instead. Or use Download theme.json / Copy CSS for manual use.
  4. 4 Reuse by name: save the json as assets/presets/custom.json, then just say "use the custom theme".
